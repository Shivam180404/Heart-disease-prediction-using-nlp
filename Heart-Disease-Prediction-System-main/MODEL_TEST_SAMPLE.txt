Heart Disease Model - Sample Test Inputs
=======================================

Use these values in the "Add Heart Detail" form to test prediction quickly.
Field order:
age, sex, cp, trestbps, chole, fbs, restecg, thalach, exang, old_peak, slope, ca, thal

Notes:
- sex: Male/Female (or 1/0)
- All other fields should be numeric.
